📖 About This Project

This Book Recommendation System is built using Machine Learning techniques to help users discover books based on their preferences.

🔍 How It Works

  • We use a Collaborative Filtering model trained on real user ratings data.
  • Each book is represented in a pivot matrix, and cosine similarity is used to find similar books.
  • The model recommends top 6 books similar to the one entered by the user.

🧠 Tools & Technologies Used

  • Python (Flask Framework)
  • Pandas, NumPy, Scikit-learn
  • HTML, CSS, Bootstrap
  • RapidFuzz for fuzzy searching (optional)

💾 Dataset

The data used comes from the Book-Crossing dataset, which includes thousands of books, users, and their ratings.

📈 Features

  • Home page shows most popular books based on ratings and votes
  • Search and get recommendations instantly
  • All books page with pagination
